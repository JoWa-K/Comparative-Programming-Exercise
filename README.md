# Student Registration Status Checker

## Purpose

This project checks a student's registration status based on the number of units.

If units are more than 7:
Overload - Approval Required

If units are 7 or less:
Registration Accepted


## How to Run

Python:
python RegistrationChecker.py

Java:
javac StudentChecker.java
java StudentChecker

JavaScript:
node Student.js

C:
gcc student.c -o student
./student

## What This Project Shows

* How to take user input
* How to use if-else statements
* How to display output
* How the same program works in different languages
