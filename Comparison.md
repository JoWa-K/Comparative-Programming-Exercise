# Comparison between languages

## Python
- It easy to write and read 
- It does not require declaring variable types
- It runs line by line

## Java
- Requires declaring variable types
- More strict since it checks the errors before the program runs 

## JavaScript
- Similar to Python (easy to write)
- Mostly used for web development
- Runs using Node.js or browser

## C
- Requires declaring all variables
- Very fast
- More difficult to write since the programmer needs to clearly define variables and follow the syntax rules

## Conclusion
All languages can solve the same problem, but python is the easiest to use, java is strict, javascript is flexible and c is fast
